# laravel-boilerplate

A simple Laravel Project with:
* Laravel-Modules
* Admin-LTE 3

## Installation

Install composer dependencies
```bash
composer install
```
Install NPM dependencies
```bash
npm install
```

create `.env` file
```bash
cp .env.example .env
```

Generate encryption key
```bash
php artisan key:generate
```

Finally run server
```bash
php artisan serve
```
